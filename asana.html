Portfolio-Level Views

All Implementations Timeline: Shows all customer projects on a single Gantt chart. Color-code by Health Status to quickly identify at-risk projects.

Workload View: See all team members' capacity across projects. Filter by date range to identify overallocation.

List View with Custom Fields: Shows Project Name, Owner, Status, Health, Phase, Go-Live Date, and Customer Tier. Sort by Go-Live Date to prioritize.

Daily Workflow

For Each New Customer:

  1. Create Project from Template - Name it "[Customer Name] Implementation"
  2. Set Custom Fields - Implementation Type, Customer Tier, Go-Live Date
  3. Customize Timeline - Adjust task dates based on customer's target
  4. Set Dependencies - Link tasks that must happen in sequence
  5. Assign Team Members - Assign implementation manager and team
  6. Add to Portfolio - Ensure visibility in portfolio view

Daily Team Workflow:

  1. Check "My Tasks" for today's assignments
  2. Update task status as work progresses
  3. Add comments for stakeholder updates
  4. Mark tasks complete when done
  5. Check Portfolio view for cross-project insights
Weekly Updates: Every Monday, Implementation Managers should:
  • Update Health Status for each project
  • Review and address any blockers
  • Post status updates in project descriptions
  • Check team workload in Portfolio view
